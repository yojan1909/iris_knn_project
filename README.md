# iris_knn_project
# Iris Flower Classification using KNN 
This is a beginner-friendly machine learning project that classifies Iris flowers into: - Setosa - Versicolor - Virginica 
## Algorithm 
K-Nearest Neighbors (KNN) 
## Dataset 
Iris dataset from scikit-learn 
## Features - Sepal Length - Sepal Width - Petal Length - Petal Width 
## Project Structure - `train_model.py` → model training script - `app.py` → Streamlit web app - `models/` → saved model and scaler - `data/` → dataset CSV - `requirements.txt` → dependencies 
## How to Run 
### 1. Install requirements 
```bash 
pip install -r requirements.txt 
2. Train the model 
python train_model.py 
3. Run Streamlit app 
streamlit run app.py 
Output 
The app predicts Iris flower species based on flower measurements. ---
